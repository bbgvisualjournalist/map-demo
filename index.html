Map demo

Here's a mockup showing how to use jQuery, TabletopJS and Leaflet to load in some external data from a Google Spreadsheet and then map it. Open the browser's console window and you can watch the data get loaded. (The code is liberally commented.)

The politicians

The outline

Here are the steps we're going to take

  1. Define the global variables
  2. Wait for the page to load
  3. Load the spreadsheet data
  4. When the data is returned, build the map
    • Define the tile set
    • Create the map (and set the initial focus)
    • Define any custom icons
    • Loop through the data to add markers
